次のような方針を考えます：

- ベルトコンベアの配置は、次で固定する：
  - 高さ 2, 幅 N のベルトコンベアを縦に n / 2 個配置する
  - 高さ N, 幅 2 のベルトコンベアを横に n / 2 個配置する

- このようにしたとき、以下のような性質があると考えています。
  - 基本的には、次に搬出するべき荷物が搬出口に近くなるのがよい
  - さらに、搬出するべき荷物に近い番号の荷物は、搬出するべき荷物と一緒に動かしてしまうのが効率が良い
  - 逆に、すでに搬出し終わり、-1 となっている荷物については、できるだけ搬出口から遠いところに置いておくのがよい

これを踏まえて、適当な評価関数を実装し、それを改善していくようにコンベアを動かす、という方針を、ざっくりと考えています。
